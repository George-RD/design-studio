Design Studio

Give it a brief. Get a website that looks like a real studio made it. Four agents split the work: one plans, one designs, one builds, one checks. They work apart. So the design never bends to fit the code.

“
Most tools make the same page over and over: a centred headline, a button, a grid of cards. It looks safe. It is also forgettable.

The trap is code-anchoring bias. When AI reads its own code first, it just tweaks what is already there. It never asks what the page should feel like. Design Studio breaks the loop. The brief, the design, the build, and the check each happen in a separate room. The result looks like a studio made it. Because one did.

How it works

Process

Four jobs. Four agents. One loop. Each agent only sees what it needs. That gap is the point: the designer never gets anchored to the code.

01
Plan Take a brief and turn it into a creative direction.
02
Design Describe the page in words, not code.
03
Build Translate the description into production code.
04
Evaluate Open the page, use it, score it, and feed back.

Then it loops. Each round scores the page and feeds the notes back. It stops when the design is good enough to ship.

The studio difference

What Makes It Different

  • 01
    Originality by separation The creative eye never reads the code. The builder never sees the brief. This isolation is what stops the work from drifting back to the default.
  • 02
    Real browser proof The evaluator opens the result in a browser, clicks, scrolls, and reads it. A page that works only in theory is not good enough.
  • 03
    Zone-by-zone judgement Every section is scored on its own. A strong hero cannot hide a broken footer.
  • 04
    Adversarial testing Before the final score, the evaluator tries to break the layout. Overflow, dead buttons, and mobile collapse are caught early.
  • 05
    Taste as a constraint Design Studio refuses to ship work that looks like a template. Originality and design quality are weighted above everything else.

The rubric

How Scoring Works

Score = (Design × 2 + Originality × 2 + Craft + Functionality) ÷ 6

Design Quality 2x
Originality 2x
Craft 1x
Functionality 1x

Get started

Install.

You need Claude Code and Chrome with the claude-in-chrome MCP server running.

› claude plugin install https://github.com/George-RD/design-studio copy

Or load it for one session: claude --plugin-dir ./design-studio

After installing, run /reload-plugins in Claude Code.

Reference

Commands

/design-studio:create <your brief> Run the full design loop from a short description.
